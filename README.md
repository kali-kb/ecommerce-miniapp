# Online Marketplace Ethiopia Miniapp

This is a mini app designed to connect buyers and sellers in a streamlined and user-friendly environment. 

## Features

* **Buyer Features:**
    * Browse Products
    * Contact Seller

* **Seller Features:**
    * List Products
    * Manage Inventory

## Technologies Used

* **Frontend:**  Vue.js
* **Backend:** Node.js with Express
* **Database:** PostgreSQL
